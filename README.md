# HANS Backend
Home AutomatioN System

Backend do serwera IoT obsługujacy protokół MQTT.
* Java 8
* SpringBoot 2.3.3
* SpringSecurity + JWT
* JPA + Hibernate
* PostgreSQL 10

Wykorzystane Feature MQTT:
* Autoryzacja node przed połączeniem z brokerem
* sub/pub od zweryfikowanych node
* broadcast danych do node
